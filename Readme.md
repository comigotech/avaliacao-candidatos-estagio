# Desafios de Programação - Avaliação de Estágio

## Visão Geral

Bem-vindo(a) ao repositório de desafios de programação! Este projeto foi desenvolvido como parte de um processo de avaliação de estágio e está organizado em múltiplas etapas, cada uma focada em uma tecnologia específica e na resolução de diferentes problemas de lógica e estrutura de dados.

## Estrutura do Repositório

O projeto está dividido em três pastas principais, uma para cada etapa da avaliação. Para entender os desafios específicos de cada etapa, bem como as instruções para configurar o ambiente e executar o código, por favor, **consulte o arquivo `README.md` dentro de cada pasta**.

---

### 📂 [Etapa 1: Desafios em Java](./Etapa1/README.md)

Nesta etapa, os desafios foram resolvidos utilizando **Java**, com foco em Lógica de Programação e conceitos de Programação Orientada a Objetos (POO).

* **Tecnologia:** Java
* **Instruções:** [Clique aqui para ver o README da Etapa 1](./Etapa1/README.md)

---

### 📂 [Etapa 2: Desafios em JavaScript](./Etapa2/README.md)

A segunda etapa aborda a resolução de problemas utilizando **JavaScript (ES6+)**, explorando a manipulação de arrays e objetos de forma funcional.

* **Tecnologia:** JavaScript (com Node.js)
* **Instruções:** [Clique aqui para ver o README da Etapa 2](./Etapa2/README.md)

---

### 📂 [Etapa 3: Desafios em TypeScript](./Etapa3/README.md)

A terceira e última etapa utiliza **TypeScript** para resolver os desafios, aproveitando recursos como tipagem estática, interfaces, classes e generics para criar um código mais robusto e seguro.

* **Tecnologia:** TypeScript (com Node.js)
* **Instruções:** [Clique aqui para ver o README da Etapa 3](./Etapa3/README.md)
